# Frontend-version-control-task-Jonathan-Ogbonna
A collaborative version-controlled project using Git and GitHub to demonstrate my understanding of version control workflows, collaboration practices, and Git commands learned.
What This Project Demonstrates:
Initialize and manage a repository
Work with branches — creating, switching, merging
Commit and push code — with clear, descriptive messages
Create and review pull requests
Merge changes effectively — resolving conflicts when needed

Branch names:
feature-header: this branch was created for the header feature
feature-footer-renamed: this branch was created for footer features. It was initially named feature-footer and later renamed to the current name.
